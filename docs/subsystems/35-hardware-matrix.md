# 硬件选型矩阵

## §0 — One-liner

硬件选型矩阵为DVAS项目的不同数据采集范式（Ego-centric、UMI、Sim2Real、遥操作）提供系统化的硬件配置方案。

### 1.3 全局快门 vs 卷帘快门

| 特性 | 全局快门 (Global) | 卷帘快门 (Rolling) |
|------|-------------------|---------------------|
| 曝光方式 | 所有像素同时曝光 | 逐行扫描曝光 |
| 运动伪影 | 无 | 有（果冻效应） |
| 成本 | 高 | 低 |
| 适用场景 | 高速运动、SLAM | 静态/低速场景 |
| 代表传感器 | Sony Pregius系列 | 大多数CMOS |
| 推荐应用 | 机器人视觉、VIO | 监控、静态采集 |

**DVAS推荐**：
- **Ego-centric采集**：全局快门（减少运动模糊）
- **UMI系统**：全局快门或高帧率卷帘快门
- **遥操作**：全局快门（精确捕捉快速操作）
- **Sim2Real**：根据仿真传感器配置选择

## §2 — IMU选型

### 2.1 消费级 vs 工业级IMU

| 参数 | 消费级 | 工业级 | 导航级 |
|------|--------|--------|--------|
| 代表型号 | MPU-9250, BMI270 | ADIS16495, STIM300 | Honeywell HG1700 |
| 加速度计噪声 | ~200 μg/√Hz | ~20 μg/√Hz | ~1 μg/√Hz |
| 陀螺仪噪声 | ~0.01 °/s/√Hz | ~0.001 °/s/√Hz | ~0.0001 °/s/√Hz |
| 零偏稳定性 | 差 | 好 | 极好 |
| 温度补偿 | 基础 | 全温区 | 全温区+校准 |
| 价格 | $5-20 | $500-5000 | $10,000+ |
| 适用场景 | 手机、可穿戴 | 机器人、无人机 | 自动驾驶、航天 |

### 2.2 IMU在DVAS中的应用场景

| 应用场景 | 推荐IMU | 精度要求 | 备注 |
|----------|---------|----------|------|
| Ego-centric SLAM | BMI270, ICM-42688-P | 中 | 与视觉融合 |
| UMI系统姿态估计 | MPU-9250, BMI270 | 低-中 | 辅助定位 |
| 遥操作臂状态估计 | ADIS16495 | 高 | 精确控制 |

## §6 — 与DVAS项目的关联

### 6.1 DVAS硬件架构建议

```
┌─────────────────────────────────────────────────────┐
│              DVAS 硬件架构建议                       │
├─────────────────────────────────────────────────────┤
│                                                     │
│  数据采集层                                          │
│  ├── Ego: Meta Aria / GoPro + Jetson Orin NX       │
│  ├── UMI: 定制夹爪 + GoPro + Arduino/ESP32          │
│  ├── Sim: NVIDIA GPU工作站 + Isaac Sim/MuJoCo      │
│  └── Teleop: ALOHA / Franka + VR                   │
│                                                     │
│  数据处理层                                          │
│  ├── 边缘: Jetson Orin (实时预处理)                 │
│  └── 云端: GPU集群 (训练、存储)                     │
│                                                     │
│  数据存储层                                          │
│  ├── 本地: NAS (原始数据)                           │
│  └── 云端: 对象存储 (备份、共享)                    │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 6.2 硬件采购优先级

**阶段1：原型验证**（预算<$2,000）
1. UMI手持设备（$400）
2. GoPro/Insta360（$500）
3. Jetson Orin Nano（$500）
4. 基础传感器套件（$200）

**阶段2：研究扩展**（预算<$10,000）
1. 双臂遥操作平台（ALOHA配置）
2. 多视角相机阵列
3. 高性能工作站
4. Meta Aria/VR设备

**阶段3：规模部署**（预算>$20,000）
1. 多台机械臂（Franka/xArm）
2. GPU训练集群
3. 专业采集工作室
4. 数据存储基础设施
