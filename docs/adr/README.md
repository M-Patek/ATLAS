# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for ATLAS.

## What is an ADR?

An ADR captures an important architectural decision made along with its context and consequences. ADRs are immutable once accepted; if a decision changes, a new ADR supersedes it.
